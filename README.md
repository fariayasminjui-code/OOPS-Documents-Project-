Digital Forensics System

A console-based Digital Forensics System developed in C++ using Object-Oriented Programming (OOP) concepts.
This project is designed to collect, preserve, analyze, and report digital evidence securely for basic cybercrime investigations.

Features
Evidence File Collection
Custom Hash Generation
Metadata Extraction
Keyword Search in Files
Activity Logging System
Chain-of-Custody Tracking
TXT Report Generation
Menu-Driven Console Interface
Technologies Used
C++
Object-Oriented Programming (OOP)
STL (vector, map)
fstream
stringstream
g++ Compiler
Project Objectives
Ensure evidence integrity using hashing
Practice real-world OOP implementation
Build a standalone forensic investigation tool
Maintain secure investigation logs
Project Scope
Local file system analysis
Offline desktop application
Basic digital forensic investigation
Extendable with GUI support in the future
System Modules
Evidence Manager

Handles adding, removing, and managing evidence files.

Hash Generator

Generates custom hash values for integrity verification.

Analysis Module

Extracts metadata and performs basic file analysis.

Logging System

Maintains investigation logs and chain-of-custody records.

Report Generator

Exports investigation reports in TXT format.
